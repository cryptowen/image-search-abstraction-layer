Image Search Abstraction Layer

User stories:
  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example usage 1:

{{ app_url }}/api/imagesearch/lolcats%20funny?offset=10
